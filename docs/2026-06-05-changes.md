잘려서(`__truncated__`) 변환을 완료하지 못하는 문제가 있었다. 주 원인:

- 리스트성 화면에서 자식 노드가 `MAX_CHILDREN`(40개)를 넘으면 뒷부분이 `__truncated__`로 생략됨
